# 测试指南

## 测试环境

### 硬件准备
- M5Stack Cardputer-Adv
- HX711 模块
- 压力传感器
- USB-C 数据线
- 已知重量的物体（如手机、杯子、砝码）

### 接线确认
```
HX711 模块          Cardputer-Adv
─────────────────────────────────
VCC (红)     →      5V
GND (黑)     →      GND
DT/DOUT (白) →      G2
SCK/CLK (绿) →      G1
```

### 软件准备
- Arduino IDE 2.x
- M5Stack ESP32 Boards
- M5Unified 库
- HX711_ADC 库

---

## 测试步骤

### 步骤 1：编译

1. 打开 Arduino IDE
2. 打开项目：`Coffee-Digital-Scale-ADV/coffee_scale.ino`
3. 选择开发板：`M5Stack Arduino` → `M5Stack-Core-ESP32-S3`
4. 点击 ✓ 编译按钮

**预期结果**：
- ✅ 编译成功，无错误
- ⚠️ 可能有警告（可忽略）

---

### 步骤 2：烧录

1. 使用 USB-C 线连接 Cardputer 到电脑
2. 点击 → 上传按钮
3. 等待烧录完成

**预期结果**：
- ✅ 烧录成功
- ✅ 屏幕自动显示初始化信息

---

### 步骤 3：初始化测试

**观察屏幕显示**：

```
时间线：
0s    → "Initializing..."
0.5s  → "Init Sensor..."
1.0s  → "Taring..."
1.5s  → "Ready!"
2.5s  → 主界面显示
```

**验证点**：
- [ ] 屏幕正常显示
- [ ] 无错误信息
- [ ] 自动去皮完成
- [ ] 主界面正常显示

---

### 步骤 4：重量显示测试

**测试 1：无负载**
1. 不放任何东西在传感器上
2. 观察重量显示

**预期结果**：
- 重量显示约 0.0g
- 显示稳定，不跳动

**测试 2：已知重量**
1. 放上已知重量的物体（如手机约 200g）
2. 观察重量变化

**预期结果**：
- 重量显示接近实际重量
- 显示稳定

**测试 3：重量变化**
1. 缓慢增加重量（如慢慢倒水）
2. 观察重量变化

**预期结果**：
- 重量实时更新
- 无明显延迟

---

### 步骤 5：自动计时测试

**测试 1：自动启动**
1. 确保重量显示为 0.0g
2. 放上超过 0.3g 的物体
3. 观察计时器状态

**预期结果**：
- 状态指示器变为 "RUNNING"（绿色）
- 计时器开始计时
- 时间显示从 00:00.0 开始

**测试 2：自动重置**
1. 移除物体
2. 等待 2 秒
3. 观察计时器状态

**预期结果**：
- 状态指示器变为 "STOPPED"
- 计时器重置为 00:00.0

**测试 3：连续计时**
1. 放上物体，开始计时
2. 保持重量，观察计时
3. 重量变化时观察计时

**预期结果**：
- 计时持续进行
- 重量变化不影响计时

---

### 步骤 6：流量计算测试

**测试 1：静态流量**
1. 保持重量不变
2. 观察流量显示

**预期结果**：
- 流量显示约 0.0 g/s

**测试 2：动态流量**
1. 缓慢倒水（约 5g/s）
2. 观察流量变化

**预期结果**：
- 流量实时更新
- 流量曲线绘制

**测试 3：快速注水**
1. 快速倒水（约 10g/s）
2. 观察流量变化

**预期结果**：
- 流量显示增大
- 流量曲线变化明显

---

### 步骤 7：页面切换测试

**测试 1：切换页面**
1. 在主界面，按键盘 '1' 键
2. 观察页面切换

**预期结果**：
- 切换到重量曲线页面
- 页面指示器更新

**测试 2：循环切换**
1. 继续按 '1' 键
2. 观察页面循环

**预期结果**：
- 主界面 → 重量曲线 → 流量曲线 → 设置 → 主界面
- 页面指示器正确

**测试 3：曲线显示**
1. 在曲线页面，进行注水操作
2. 观察曲线绘制

**预期结果**：
- 重量曲线显示重量变化
- 流量曲线显示流量变化

---

### 步骤 8：串口调试测试

**打开串口监视器**：
```
工具 → 串口监视器 → 波特率 115200
```

**预期输出**：
```
=== Coffee Digital Scale ===
Version: 1.0.0
Storage initialization failed - continuing without SD
HX711 initialized
Tare complete
Calibration factor: 420.00
Initialization complete
```

**验证点**：
- [ ] HX711 初始化成功
- [ ] 无 SD 卡不影响运行
- [ ] 使用默认校准因子
